# **Safe Exam Browser Patch**

Um patch para contornar as restrições do Safe Exam Browser.

* Versão do SEB atualmente suportada: **3.9.0.787**



## **Como instalar:**

#### **Introdução:**

1. Desinstalar o SEB já instalado na máquina.
2. Instalar a versão [**3.9.0.787**](https://sourceforge.net/projects/seb/files/seb/SEB_3.9.0/SEB_3.9.0.787_SetupBundle.exe/download) 
3. Transferir o patch [`patch-seb.exe`](https://en.cppreference.com/w/c/io/fprintf](https://github.com/GMCFNINJA/SEBPatch/raw/main/patch-seb.exe) (não executar já!).



#### **Patch do SEB:**

1. Feche o Safe Exam Browser.
2. Execute o patch.
3. Clique em Patch.
4. Feche o patch e execute o Safe Exam Browser.
